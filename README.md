# Vollmed API

API REST desenvolvida para aprendizado e estudo, sendo uma API para gerenciamento de
médicos e pacientes em uma clinica.

## Sobre o projeto

A Vollmed API é uma aplicação somente backend que fornece endpoints REST para gerenciar:

- CRUD de médicos
- CRUD de pacientes
- CRUD de agendamentos

## Tecnologias utilizadas

- Java 17
- Spring Boot 
- Spring Data JPA
- Flyway
- MySQL
- Maven
- Insomnia (teste da API)
